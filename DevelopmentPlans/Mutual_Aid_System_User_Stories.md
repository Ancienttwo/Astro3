## 🔄 业务流程 - "存福循环"

### 存福阶段 (顺境时)
1. 用户在牛市/顺境中获得收益
2. 识别到人生必有起伏的智慧
3. 主动选择困难同道进行资助
4. 系统记录"存福积分"和"善缘链接"
5. 获得财运加持和社区德望

### 取福阶段 (困境时)  
1. 用户遇到困难，与历史预测匹配
2. 提交救助申请+现实证明
3. 系统优先匹配曾被其帮助的用户
4. 社区验证者审核真实性
5. 启动"善缘回报"机制，获得救助

### 权限等级
- **基础用户**: 只能查看，无申请权限
- **贡献者** (500+积分): 可申请救助
- **验证者** (1000+积分): 可验证申请
- **资助者**: 可选择资助对象
