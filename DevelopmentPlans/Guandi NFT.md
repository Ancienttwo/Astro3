## 产品需求文档 (PRD): 关帝灵签 NFT - “运气即挖矿”

**版本**: 1.0  
**日期**: 2025年8月5日

### 1. 概述

“关帝灵签 NFT”是一个融合了中华传统文化、游戏化激励和 Web3 资产所有权的核心功能。用户通过模拟真实的求签和“掷圣杯”过程，获得每日的运势指引，并将这份独特的“运气”转化为可量化的 `$luck` 积分。积分将与未来的 `$AZI` 代币空投挂钩，同时，用户可以将最心仪的灵签铸造 (Mint) 为永久的、可收藏的 NFT 资产。

本功能旨在成为 AstroZi Web3 版本的核心增长引擎，通过创新的 **“运气即挖矿” (Luck-to-Earn)** 模式，吸引、激活并留住高质量的 Web3 用户。

### 2. 目标与成功指标

| 目标 | 成功指标 |
| :--- | :--- |
| **用户增长 (Acquisition)** | 上线首月，每日新增钱包连接数 (New Daily Wallets) > 500 |
| **用户激活 (Activation)** | 新用户次日留存率 > 40% |
| **用户留存 (Retention)** | 功能上线后，产品整体 DAU/MAU > 20% |
| **商业化 (Monetization)** | 每日 NFT Mint 数量 > 50 次 |
| **社区热度 (Virality)** | Twitter `#AstroZiLuck` 话题每日新增推文 > 100 条 |

### 3. 用户流程 (User Journey)

1.  **进入页面**: 用户进入“关帝灵签”功能页。
2.  **开始求签**: 点击“开始今日求签”按钮（每日仅限一次有效开始）。
3.  **掷圣杯**:
    *   用户点击“掷圣杯”按钮。
    *   **结果 A (一正一反)**: 成功！连续成功次数 +1。如果连续成功 3 次，进入步骤 4。
    *   **结果 B (正正)**: “神明在笑”，提示用户重新掷杯。不消耗每日机会，连续成功次数归零。
    *   **结果 C (反反)**: “缘分未到”，今日求签机会结束。提示用户明日再来。
4.  **获得灵签**: 成功后，展示精美的灵签页面，包含签文、解说和获得的 `$luck` 积分数量。
5.  **后续操作**:
    *   用户可将灵签保存到“我的灵签”画廊。
    *   用户可选择支付费用，将此灵签 Mint 成 NFT（每个钱包最多 3 次）。
    *   用户可将求签结果或 Mint 的 NFT 分享到社交媒体。

### 4. 功能需求 (Functional Requirements)

#### 4.1. 求签核心模块

*   **每日机会限制**: 每个钱包地址每天只能成功**开始**一次求签流程。`正正`导致的重掷不计入次数。
*   **状态管理**: 系统需记录用户当天的状态：`未开始`, `进行中`, `已成功`, `已失败`。
*   **防作弊**: 后端需验证求签流程的合法性，防止无限次求签。

#### 4.2. 掷圣杯小游戏

*   **UI/UX**: 必须有强烈的仪式感。包括一个虚拟的签筒和一对圣杯。
*   **动画效果**:
    *   点击按钮后，圣杯从签筒中飞出、旋转、落地的流畅动画。
    *   结果揭晓时，应有对应的视觉和音效反馈（如成功时的金光、失败时的暗淡）。
*   **逻辑**: 后端生成随机结果，前端负责展示。`一正一反`的概率应略高于 50%，以保证基本的游戏体验。

#### 4.3. $luck 积分系统

*   **积分发放**: 成功获得灵签后，根据签的等级（上上、上吉、中吉等）发放不同数量的 `$luck` 积分。
*   **积分显示**: 在用户个人中心和求签页面的显著位置，实时显示用户累积的 `$luck` 积分总额。

#### 4.4. NFT 铸造 (Mint)

*   **“我的灵签”画廊**: 用户求到过的所有签文都会被记录在这里，方便回顾和选择。
*   **Mint 按钮**: 每个未被 Mint 的灵签旁都有一个“铸造成 NFT”的按钮。
*   **付费流程**: 点击 Mint 后，调用钱包进行支付（如 MATIC, ETH 等）。
*   **次数限制**: 后端和智能合约共同限制每个钱包地址最多只能 Mint 3 次。
*   **智能合约**:
    *   标准的 ERC-721 合约。
    *   Metadata 应包含：签文编号、等级、完整签文、求签日期、以及独特的 AstroZi 品牌标识。
    *   NFT 图片应是动态或高质量的静态图，设计精美。

### 关帝灵签增长飞轮：一个“运气即挖矿” (Luck-to-Earn) 的新范式

你的核心设计是一个自我强化的循环，我们可以将其定义为 **“运气飞轮”**。



#### **飞轮一：吸引 (Acquire) - “求个好签，博个空投”**

这是用户进入飞轮的入口。

*   **核心驱动力**:
    1.  **文化吸引力**: “关帝灵签”本身对华人用户有极强的文化认同和情感连接。
    2.  **空投预期**: `$luck` 积分与 `$AZI` 代币空投直接挂钩，这是吸引 Web3 用户的最强诱饵。
    3.  **社交炫耀**: 求到“上上签”或稀有签，本身就是一种社交资本。

*   **运营策略**:
    1.  **市场定位**: 宣传口号定为 **“首个‘运气即挖矿’(Luck-to-Earn) 的玄学 DApp”**。强调你的运气可以直接转化为未来价值。
    2.  **内容营销**: 在 Twitter、Discord、小红书等平台，发布关于“如何求得上上签”、“不同灵签的现代解读”等内容，吸引泛文化用户。
    3.  **KOL 合作**: 与玄学、Web3 领域的 KOL 合作，让他们直播求签过程，分享求到的好签，并强调 `$AZI` 空投的潜力。

#### **飞轮二：激活与参与 (Engage) - “每日一掷，与天对赌”**

这是飞轮转动的核心引擎，目标是让用户每天都回来。

*   **核心驱动力**:
    1.  **每日稀缺性**: “每天一次”的限制，创造了稀缺性，让用户珍惜每一次机会。
    2.  **游戏化机制**: “掷圣杯”的设计非常巧妙：
        *   **正正 (软失败)**: 重新开始但不消耗次数。这是一种“再来一局”的心理钩子，增加了单次会话的停留时长和趣味性，避免了用户因一次失误就立刻流失。
        *   **反反 (硬失败)**: 机会丢失。这制造了“损失厌恶”心理，强化了第二天再来的动机。
        *   **连续三次成功**: 这是一个有挑战性的目标，完成时会带来巨大的成就感。
    3.  **即时奖励**: 成功后立即获得 `$luck` 积分，正反馈闭环非常短。

*   **运营策略**:
    1.  **优化体验**: 掷圣杯的过程一定要有仪式感。精美的动画、清脆的音效、带有悬念的停顿，都能极大增强用户粘性。
    2.  **推送提醒**: 通过浏览器或 App 推送，提醒用户“今日的求签机会已刷新，快来试试手气吧！”
    3.  **建立排行榜**: 设立 **“$luck 积分榜”** 和 **“天选之人榜”** (连续成功次数最多或求到上上签最多的用户)，利用竞争心理驱动日活。

#### **飞轮三：留存与变现 (Retain & Monetize) - “铸造好运，分享价值”**

这是飞轮的加速器，将短期用户转化为长期信徒和付费用户。

*   **核心驱动力**:
    1.  **长期投资**: 累积的 `$luck` 积分成为用户的“沉没成本”，积分越多，他们对 `$AZI` 空投的期望就越高，越不可能离开。
    2.  **所有权与收藏**: 将心仪的签文 Mint 成 NFT，满足了用户“拥有好运”的心理需求。这是从“体验”到“资产”的关键转化。
    3.  **稀缺性驱动付费**: “最多只能 Mint 三支”的限制，让用户必须精挑细选，增加了决策的郑重性和 NFT 的珍贵性。

*   **运营策略**:
    1.  **NFT 设计**:
        *   NFT 卡面设计要精美，具有东方美学和赛博朋克结合的风格。
        *   卡面应包含：**签文编号、吉凶等级、签文本身、动态背景、以及独一无二的 Mint 编号**。
        *   可以推出**稀有签文 NFT**，例如“上上签”有特殊的金色边框或动画效果。
    2.  **付费点设计**:
        *   Mint NFT 的费用不宜过高，可以设定为 5-10U 的价格，降低参与门槛。
        *   可以考虑动态定价，越稀有的签文 Mint 费用越高。
    3.  **建立个人画廊**: 为每个用户创建一个“我的灵签”页面，展示他们求到过的所有签文和已 Mint 的 NFT。这是一个强大的留存工具。

#### **飞轮四：推荐 (Refer) - “分享我的 NFT，邀请你来求签”**

这是飞轮实现自增长的关键，让现有用户自发地带来新用户。

*   **核心驱动力**:
    1.  **社交证明**: 在社交媒体上分享自己 Mint 的“上上签”NFT，是一种非常有效的“凡尔赛”，能直接触发朋友的好奇心。
    2.  **内置邀请**: NFT 本身就是最好的广告。

*   **运营策略**:
    1.  **一键分享**: 在用户成功 Mint NFT 后，提供一个“分享到 Twitter/Discord”的按钮。
    2.  **预设分享文案**: 自动生成文案，如：“刚在 @AstroZi 求到了第88号上上签，并铸造成了我的专属幸运 NFT！你也来试试手气，一起赚取 $AZI 空投吧！#LuckToEarn #NFT #关帝灵签”
    3.  **推荐奖励**: 如果分享链接带来了新用户，并且该新用户成功求签或 Mint 了 NFT，原分享者可以获得额外的 `$luck` 积分奖励。这就把社交分享变成了有明确回报的推荐行为。
